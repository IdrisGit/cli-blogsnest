## Getting Started

#### Prerequisites

- [Node.js](https://nodejs.org/) >=18.0
- [NPM](https://www.npmjs.com/)

### Try it now

Run

```
npx cli-blogsnest
```

### Local

#### Installation

1. Clone the repository:

```
git clone git@github.com:IdrisGit/cli-blogsnest.git
```

2. Navigate to the project directory:

```
cd cli-blogsnest
```

3. Install dependencies:

```
npm install
```

#### Usage

```
node index.js
```
